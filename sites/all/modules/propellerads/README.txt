
### Introduction

This module serves for integrate Propeller Ads popups in your Drupal only activating and configuring this module.

### Setup procedure

Copy under sites/all/modules and go to /admin/modules. Check it and enable it.

Alternatively you can download and install it using Drush with using the following command:

<pre>
drush -y en propellerads
</pre>
<p></p>

Later you should go to /admin/config/propellerads and introduce your Publisher and Channel ID.
